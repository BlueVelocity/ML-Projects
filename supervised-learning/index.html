This program is built using p5js and displays the functional training of a Perceptron, the basic building block of Universal Function Approximators and a key component in machine learning models.

The program represented above is designed to guess whether or not a each of a set of points are above or below the line, and uses a form of supervised learning to correct for the error between the guess and the known coordinates to approximate the equation of the line.

The correct line equation is displayed by the thin black line, while the Perceptrons guess at the correct equation is displayed in blue.
